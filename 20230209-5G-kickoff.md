- 장소: ETRI 7-462
- 참석

  - 윈스: 주은영,박철정
  - 시스메이트: 최간호

## 3차년도 수행 내용
### 사업계획서 상 목표 내용
- 고속 스위칭 기반 실시간 패킷 수집 및 위협 분석 HW 플랫폼 상용시제품 개발 (시스메이트)
  - 250Gbps급 패킷 수집 및 위협 분석 하드웨어 플랫폼 개발 및 성능 검증
  - 10GE 4포트 지원하는 RPS(Receive Packet Steering) 기능 구현
  - DPDK 기반 SR-IOV VSF 가상화 기능 구현
  - 80Gbps급 Secured MEC-DP 하드웨어 플랫폼 개발
  - 네트워크 위협 분석 엔진(cIPS) 통합 및 성능 검증
- 컨테이너 기반 고성능·경량 VSF 상용시제품 개발 (윈스)
  - VoIP 및 QUIC 트래픽용 cIPS 프로토타입 개발
  - 20Gbps급 5G N3 트래픽용 cIPS 상용시제품 개발
  - 10Gbps급 경량 방화벽(cFW, cWAF) 상용시제품 개발
- 5G 기반 스마트공장 보안 실증 테스트베드 구축

## 3차년도 수행 세부 사항
### 1) 윈스-5G 엣지 보안 시스템 상용시제품 개발
- 고속 스위칭 기반 실시간 위협 분석용 상용시제품 개발
- 컨테이너 기반 고성능/경량 VSF 프로토타입 개발
#### 1-1) 고속 스위칭 기반 실시간 위협 분석용 상용시제품 개발
- MEC-DP 하드웨어에 네트워크 위협 분석 엔진 (IPS) 통합
- 성능 검증을 위한 공인 시험
  - MEC-DP IPS 트래픽 처리성능: 60Gbps 양방향
  - Telco-MEC IPS 트래픽 처리성능: 200Gpbs 양방향
- 자원 사용 최적화
  - 가상화 IPS 제품의 자원 사용율 최적화 (메모리 경량화)
#### 1-2) 컨테이너 기반 고성능/경량 VSF 프로토타입 개발
- VoIP 및 QUIC 트래픽용 cIPS 프로토타입 개발
  - cIPS 엔진에 VoIP 모듈 탑재 (라이센스 기반 동작)
  - 기능 검증용 QUIC 프로토타입 개발
- 15Gbps 급 cIPS 상용시제품 개발
  - cIPS 자원(메모리/core) 경량화를 통한 상용시제품 개발
  - 가상화 라이센스 관리 기능 개발
  - 국정원 보안정책 배포 기능 개발 (TMS-PCRE Cloud)
  - 퍼블릭 클라우드 상품 등록을 위한 PoC 수행
- 10Gbps 급 cFW, cWAF 상용시제품 개발
- 클라우드 상품 등록
  - KT/NHN/Naver/KAKAO
  - **[제안]** 상용화 실적을 위해 MEC-DP 기반 IPS 제품화 제안
  - **[요청]** 성과 실적 관련 문의에 대응하기 위한 정리본 요청
  - **[요청]** 연차보고서에 해당 내용이 강조될 수 있도록 요청

### 2) 시스메이트-고속 스위칭 기반 실시간 패킷 수집 및 위협 분석 HW 플랫폼 상용시제품 개발
- 250Gbps 급 5G Telco MEC용 HW 플랫폼
- 80Gbps급 Secured MEC-DP 하드웨어 플랫폼
- 200Gbps급 패킷 수집 위협분석 플랫폼
- 60Gbps급 Secured MEC-DP 플랫폼
#### 2-1) 250Gbps 급 5G Telco MEC용 HW 플랫폼
- 100GE 인라인 2 회선 지원 최대 400Gbps 대역폭 수용 가능 시스템 개발
  - 상용 100GE MAC 칩셋과 DPDK 오픈 소프트웨어 기술을 접목하여 성능 확보 계획
  - 시스템 장애 대비 FOD (Fail Over Device) 기능 개발
#### 2-2) 80Gbps급 Secured MEC-DP 하드웨어 플랫폼 개발
- 10GE 인라인 8 회선 지원 최대 80Gbps 대역폭 수용 가능 시스템 개발
  - 상용 10GE MAC 칩셋과 DPDK 오픈 소프트웨어 기술을 접목하여 성능 확보 계획
  - NIC 에 내장된 McP 를 이용한 5G N3 LBO(Local Breakout) 기능 구현
#### 2-3) 200Gbps급 패킷 수집 위협분석 플랫폼 개발
- 200Gbps 성능을 지원하는 패킷 수집 및 하드웨어 상용시제품 제작
  - Rack-mountable 2U 네트워크 어플라이언스, 100GE 인라인 NIC 2장 장착
  - 공동 연구 기관 (윈스) IPS 200Gbps 위협분석 성능 지원을 위한 지원
- DPDK 를 이용한 시험용 어플리케이션으로 200Gbps 성능 검증 완료
- IPS 위협 분석 시스템 연동 성능 시험 지원 완료 (ETRI 성능 시험)
#### 2-4) 60Gbps급 Secured MEC-DP 플랫폼 개발
- 60Gbps Secured MEC-DP 시스템 상용 시제품 제작
  - 10GEMEC-DP NIC을 네트워크 어플라이언스에 호환 장착되도록 개발
  - 공동 연구 기관 (윈스)와 DPDK를 이용한 연동 완료
  - 60Gbps MEC-DP 성능 지원을 위한 지원
- Secured MEC DP NIC 오프로드 엔진 성능 시험
  - DPDK 시험 어플리케이션, STC 시험기를 이용한 60Gbps MEC DP 오프로드 성능 시험
- 80Gbps 급 Secured MEC DP 하드웨어 플랫폼 연동 완료
  - IP 및 port 에 따라 DROP/LBO/BYPASS/UPLOAD action 지원
  - INLINE/MIRROR 모드를 비롯한 다양한 라우팅 기능을 제공
- cIPS 컨테이너 기반 위협 분석 시스템 연동 및 경산 테스트베드 시험 완료
- 5G SMF 와 PFCP(Packet Forwarding Control Protocol) 연동 기능 개발
  - Secured MEC DP UPF 의 Rule 관리 기능 개발
- UERANSIM, Open5GS 시뮬레이션 환경 시험
